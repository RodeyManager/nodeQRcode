<h1>二维码生成器</h1>
<p>采用 node-webkit 开发的一个二维码生成器</p>

<article>
	<h2>使用：</h2>
	<p>
	开发使用之前你必须先下载<strong>node-webkit</strong>, 然后将webapp.zip拖到nw.exe上运行。如果想直接运行exe文件来打开应用程序，
	你需要使用exe打包工具，这里推荐使用Enigma Virtual Box。需要注意的是，打包时nw.exe和icudtl.dat文件是必须包含的。
	</p>
